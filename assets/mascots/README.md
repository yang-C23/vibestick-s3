# Mascots

**Original characters only — do NOT add official logos or trademarked icons** (no Anthropic/Claude,
OpenAI/Codex, or other brand marks). Bundling those would create trademark/IP problems for an
open-source project.

Suggested original personas (placeholders until art lands in M6):

- **Codex** → a friendly robot or owl.
- **Claude** → a fox, capybara, or bookworm.
- **Terminal / generic** → a blinking cursor sprite.

## Per-state frames

Each pack provides small sprites (or 1-bit bitmaps for the 135×240 screen) for:
`idle` (sleep) · `recording` (ears-up) · `transcribing` (typing) · `running` (tapping) ·
`needs_approval` (raised hand) · `done` (celebrate) · `failed` (confused).

## Custom packs

Drop your own art in `assets/mascots/<pack-name>/` and select it via the device `mascotPack` config.
By contributing art here you license it under the repository's MIT license (original work only).
